# More Free Tools

Marcus Quinn (mabo-du) develops a wide range of open-source tools for archaeologists, historians, and heritage professionals. Below is a list of other free, public applications and libraries that you might find useful:

## 🏛️ Archaeological Data & Excavation
- **[HOARD](https://github.com/mabo-du/HOARD)**: Heritage Object and Artifact Research Database — an AI-assisted archaeological pipeline for document ingestion, OCR, analysis, and report generation.
- **[trowel](https://github.com/mabo-du/trowel)**: Transforms digital excavation data into compliance-ready archaeological reports. Upload your context records, finds catalogues, and sample logs to get a professionally structured draft report in seconds.
- **[stratigraph](https://github.com/mabo-du/stratigraph)**: A Modern, AI-Ready Harris Matrix Generator & Digital Heritage Hub.
- **[acta-523](https://github.com/mabo-du/acta-523)**: Free, open-source DPR 523 compliance form builder for CRM archaeology. Generates SHPO-ready PDFs offline.
- **[dibble](https://github.com/mabo-du/dibble)**: Digital Image-Based Benchmark for Lithic Evaluation (photogrammetry + 3D lithic analysis).

## 🔬 Geomatics & Scientific Dating
- **[lidar-relief-qgis-plugin](https://github.com/mabo-du/lidar-relief-qgis-plugin)**: A QGIS Processing plugin for advanced archaeological terrain visualization from Digital Elevation Models (DEMs).
- **[rubipont](https://github.com/mabo-du/rubipont)**: A high-performance Rust library and CLI tool for lossless conversion between major LiDAR point cloud formats.
- **[libby](https://github.com/mabo-du/libby)**: Modern radiocarbon calibration with LLM-powered narratives, Bayesian modelling, and GIS curve selection.
- **[fritts](https://github.com/mabo-du/fritts)**: An open-source desktop dendrochronology tool for tree-ring cross-dating, measurement, and master chronology building.
- **[IsoMap](https://github.com/mabo-du/IsoMap)**: A desktop application that automates the standardisation, validation, and export of legacy isotopic and paleoecological datasets for submission to centralised repositories (Neotoma, IsoArcH).

## 📜 Archives & Oral History
- **[lore](https://github.com/mabo-du/lore)**: A desktop application designed for historians, archivists, and researchers providing state-of-the-art AI transcription, speaker diarization, named entity recognition, and translation—100% offline, on your own hardware.
- **[cartulary](https://github.com/mabo-du/cartulary)**: Spreadsheet to EAD3 XML Finding Aid Converter.

## 💻 Developer Tools
- **[heritage-types](https://github.com/mabo-du/heritage-types)** / **[heritage-cli](https://github.com/mabo-du/heritage-cli)**: Core infrastructure and types for the heritage software ecosystem.
- **[pluck](https://github.com/mabo-du/pluck)**: Paste a git URL, auto-detect the install method, auto-install, and you're done!
