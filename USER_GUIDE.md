# DIG User Guide

Welcome to **DIG (Digital Imaging for Geophysics)**. 

The full user manual, API references, processing guides, and format specifications are hosted online via our MkDocs site.

Please visit the [Documentation Site](https://mark.github.io/DIG) to view the User Guide.

Alternatively, you can build and view the documentation locally:
```bash
pip install mkdocs-material mkdocstrings[python]
mkdocs serve
```
Then open `http://localhost:8000` in your web browser.
